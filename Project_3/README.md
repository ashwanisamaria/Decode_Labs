# 🚀 AI Career Recommendation System

An intelligent career recommendation system built using **Python** and **Streamlit** that suggests suitable career roles based on the skills entered by the user.

This project uses **TF-IDF Vectorization** and **Cosine Similarity** to match user skills with predefined job roles.

---

## 📌 Features

* Skill-based career recommendation
* Fast and simple UI using Streamlit
* Uses Machine Learning concepts for similarity matching
* Real-time recommendation system
* Beginner-friendly AI project

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-learn
* TF-IDF Vectorizer
* Cosine Similarity


---

## 🎯 How It Works

1. User enters their skills.
2. System converts all skills into TF-IDF vectors.
3. Cosine similarity is calculated.
4. Best matching career roles are recommended.
