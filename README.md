# 🤖 Rule-Based AI Chatbot

A simple **Rule-Based Artificial Intelligence Chatbot** built using Python as part of the DecodeLabs Industrial Training Program.

This chatbot is designed to simulate human conversation using predefined rules and logic. It demonstrates the basic concepts of Artificial Intelligence such as **control flow, decision making, dictionaries, loops, and input processing**.

---

## 📌 Project Overview

This project is the foundation of learning AI before moving into advanced concepts like:

- Machine Learning
- Deep Learning
- Natural Language Processing
- Generative AI

The chatbot works by taking user input, matching it with predefined responses, and returning the appropriate output.

---

## 🚀 Features

✅ Greeting Responses  
✅ Date and Time Support  
✅ Joke Generator  
✅ Help Command  
✅ Exit Command  
✅ Chat History Storage  
✅ Unknown Input Handling  
✅ Input Sanitization (`lower()` and `strip()`)  
✅ Dictionary-Based Response Mapping
