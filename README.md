# 🤖 Decode Labs AI Chatbot

A Rule-Based AI Chatbot built using **Python**, **Flask**, and **HTML** as part of the DecodeLabs Industrial Training Program.

This chatbot simulates human-like conversation using predefined rules and dictionary-based response mapping. It demonstrates the core concepts of Artificial Intelligence such as control flow, decision-making, input sanitization, and chatbot architecture.

## 📌 Project Overview

This project is designed to build a strong foundation in Artificial Intelligence before moving into advanced topics like:

- Machine Learning
- Deep Learning
- NLP (Natural Language Processing)
- Generative AI

The chatbot accepts user input, processes it, and returns predefined responses based on matching rules.

---

## ✨ Features

- Greeting Responses
- Help Command
- Exit Command
- Unknown Input Handling
- Input Sanitization (`lower()` and `strip()`)
- Dictionary-Based Intent Mapping
- Web Deployment with Flask
- Render Hosting Support

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/ashwanisamaria/Decode_Labs.git
```

Move into the project folder:

```bash
cd Decode_Labs
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```

Open in browser:

```bash
http://127.0.0.1:5000
```
