# 🌸 Iris Flower Classification using KNN

This project is a Machine Learning classification model built using the **K-Nearest Neighbors (KNN)** algorithm.  
The model predicts the species of an iris flower based on its physical measurements.

---

## 📌 Project Overview

The goal of this project is to classify iris flowers into three categories:

- Setosa
- Versicolor
- Virginica

This is done using supervised learning with the **Iris Dataset**.

---

## 🚀 Features

- Load and preprocess dataset
- Split data into training and testing sets
- Apply feature scaling
- Train KNN classifier
- Predict flower species
- Evaluate model performance

---

## 🛠 Technologies Used

- Python
- Scikit-learn
- NumPy
- Pandas

---

## 📂 Dataset

This project uses the **Iris Dataset**.

Dataset Source (Kaggle):  
https://www.kaggle.com/datasets/uciml/iris

## 📊 Workflow

1. Import dataset  
2. Split dataset into train and test data  
3. Scale the features  
4. Train KNN model  
5. Predict test data  
6. Evaluate accuracy  
7. Generate confusion matrix and classification report

## 📈 Model Evaluation

The model is evaluated using:

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1 Score

---

## 🧠 Algorithm Used

### K-Nearest Neighbors (KNN)

KNN works by finding the nearest neighbors of a data point and predicting based on majority voting.
